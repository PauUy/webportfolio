HTML

(Hypertext Markup Language) is the standard markup language used to create web pages. It provides a set of tags and attributes that are used to structure and format content, such as text, images, and multimedia, on a web page. HTML is the foundation of the web and is used alongside other web technologies.

BootStrap

Bootstrap is a popular front-end framework used to design and develop responsive, mobile-first websites and web applications. It provides a collection of CSS and JavaScript components, such as navigation bars, forms, and more, that can be easily customized into any project. Bootstrap also offers a responsive grid system and typography.

Node.js

Node.js is an open-source, cross-platform runtime environment built on Chrome's V8 JavaScript engine. It allows developers to build server-side applications using JavaScript and enables them to use the same programming language on both the front-end and back-end of web applications.

MongoDB

MongoDB is a popular NoSQL database management system that provides high performance, high availability, and automatic scaling. It uses a flexible document model to store data in a JSON-like format, making it easy to work with for developers. MongoDB is widely used in web applications, mobile applications, and real-time analytics.
